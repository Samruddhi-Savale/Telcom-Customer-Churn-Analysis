Telcom Customer Churn Analysis is a data analysis project that studies customer behavior in a telecom company to understand why customers leave (churn).  
The project uses Python and data visualization to find patterns, trends, and key factors that influence customer churn.

---

## About This Project

This project works with a telecom customer dataset and focuses on understanding churn through data analysis and visual insights.

The main goals of this project are:

- Understand customer churn behavior
- Identify factors affecting churn
- Analyze customer demographics and services
- Visualize churn trends using charts and graphs
- Help businesses take better retention decisions

All analysis and visualizations are done using **Python libraries only**.

---

## Key Analysis Covered

- Overall churn distribution
- Churn by gender and senior citizen
- Churn based on contract type
- Relationship between tenure and churn
- Monthly charges vs churn
- Service usage impact on churn

Clear graphs are used to make the analysis easy to understand.
